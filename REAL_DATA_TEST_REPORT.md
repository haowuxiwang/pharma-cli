## 四、发现并修复的问题

### 4.1 问题 1：异常值检测失败

**现象**：运行 `pharma-cli outlier` 时 R 脚本报错
**错误信息**：`错误于round(outliers, 4): 数学函数中用了非数值参数`
**原因**：`outlier.R` 中 `round(outliers, 4)` 在处理空向量时失败
**修复方案**：添加空向量检查
```r
# 修复前
outliers = round(outliers, 4)

# 修复后
outliers = if (length(outliers) > 0) round(outliers, 4) else c()
```
**修复状态**：✅ 已修复

### 4.2 问题 2：响应面设计失败

**现象**：运行 `pharma-cli doe response_surface` 时 R 脚本报错
**错误信息**：`错误于match.names(clabs, names(xi)): 名称同原来已有的名称不相对`
**原因**：`doe.R` 中 `rbind()` 合并数据框时列名不匹配
**修复方案**：在 `rbind()` 前为所有数据框设置列名
```r
# 修复前
factorial_points <- expand.grid(replicate(n_factors, c(-1, 1), simplify = FALSE))
center_points <- as.data.frame(matrix(0, nrow = 5, ncol = n_factors))
axial_points <- as.data.frame(axial_points)
design_matrix <- rbind(factorial_points, center_points, axial_points)

# 修复后
factorial_points <- expand.grid(replicate(n_factors, c(-1, 1), simplify = FALSE))
colnames(factorial_points) <- factor_names
center_points <- as.data.frame(matrix(0, nrow = 5, ncol = n_factors))
colnames(center_points) <- factor_names
axial_points <- as.data.frame(axial_points)
colnames(axial_points) <- factor_names
design_matrix <- rbind(factorial_points, center_points, axial_points)
```
**修复状态**：✅ 已修复
